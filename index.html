读取出的数据为Unicode编码，如需计算需要强制类型转化后实现。

反思与总结

读取表格中的数字进行运算时，会出现错误，提示提取出的数字为Unicode编码，无法与int型进行相加，这是可以使用强制类型转化，用float括起来。

输入表格的中文要进行Unicode编码转化，网页中的则不需要。因为网页解析器提取出的字符自动转化。转化方式如workbook.write(u’string’)

在试验中，我觉得最重要的是找到一个正确的方向，使用正确的工具实现相应的一些功能。

关于后续拓展

在这个实验中，我使用的是直接使用firefox来进行操作，我们也可以下载其他的无界面浏览器如phantomjs等来进行操作，只需引入路径，在进行后续操作即可。可以提高一些速度。

有一些重复的功能，可以编写函数来进行实现。

html+css+js简介

HTML

HTML，超文本标记语言，主要通过标签来对网页内容进行标记，从而形成不同的模块，包括<head>和<body>两大部分。<head>中包括一些定义和链接，<body中包括网页的主要内容。

css

层叠样式表，主要用来定义网页的样式，失望也呈现出更加美观的样式和更好的用户体验。

js

在这个案例中，使用了js来进行表单的验证，js用来和用户进行交互，给网页增加一些动态效果。是一种在网页中广泛使用的脚本语言。

网页编写

网页编写过程中，主要是要填好结构，之后再做进一步的对样式的改变。


大致流程

1.首先安装好webstorm,创建一个project。

2.下载bootstrap代码，导入到project中。

3.根据要编写的内容，编写html代码，并且利用css和bootstrap组件等进行网页的一些布置和渲染。

4.最后进行一些测试，进一步调整网页结构等。

遇到的一些问题及其解决方法

  • 同时在两个窗口中输入hexo server，显示端口已经被使用，解决方法：重新分配一个窗口,例如：hexo server -p 5000
  • 在标签更新的时候，因为没有及时clean的原因，标签数量显示出现了问题。
  • 在文件部署中，要注意每个冒号后面都要留空格，否则会出现错误。
  • “关于”页面中布置的网易音乐播放器，在本地显示上传后却无法在网页显示的问题。解决，网易云音乐生成的外链播放器中使用的是http,将http改为https后，可以在网页上显示。
