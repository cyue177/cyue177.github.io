unity3d实现人物的简单控制

发表于 2016-10-28   |  

unity3d界面了解

Hierarchy 层次视图

包含每一个当前场景所有游戏对象，其中一些是资源文件的实例，如3D模型和其他预制物体（Prefab）的实例。您可以在层次结构视图中选择对象或者生成对象。当在场景中增加或者删除对象，层次结构视图中相应的对象则会出现或消失。

Scene View 场景视图

场景视图（Scene View）是交互式沙盒，可以使用它来选择和布置环境、玩家、摄像机、敌人和所有其他游戏对象。

Project 工程视图

每个Unity的项目包含一个资源文件夹。此文件夹的内容呈现在项目视图。这里存放着你游戏的所有资源，比如场景、脚本、三维模型、纹理、音频文件和预制组件。如果你在项目视图里右击任何资源，你都可以在资源管理器中（在Mac系统中是Reveal in Finder）找到这些真正的文件本身。

Inspector检视面板

显示当前当前选定的游戏对象的所有附加组件(脚本属于组件)及其属性的相关详细信息。

一些视图等

如gameview视图，可以用来观察运行状态等情况。

可参考unity3d界面功能操作详解

unity安装和配置

unity3d4.6.0破解版下载地址

  • 安装unity3d，一路next。
  • 破解，打开下载的unpch95_unityCrack文件夹，拷贝Unity 4.x Pro Patch.exe到unity安装目录下Editor文件夹下面，然后运行，点Patch按钮，读完条之后再点Ce Lic按钮， OK，破解完成。
  • 配置，打开破解好的unity3d，点Edit选项卡，选择Preferences…找到External Tools，会看到External Script Editor是Monodevelop， 点Browser选择自己喜欢的Editor。
  • 资源包导入 到unity里面选择Assets选项卡，点选import package ->custom package… 然后选中下载的资源包就可以了。
