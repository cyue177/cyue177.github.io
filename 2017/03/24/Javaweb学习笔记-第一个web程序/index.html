WEB-INF目录

  • WEB-INF目录是java的Web应用的安全目录。客户端无法访问。只有服务器端可以访问。
  • web.xml，项目部署文件
  • classes文件夹，用来存放*.class文件
  • lib文件夹，用于存放所需要的jar包
