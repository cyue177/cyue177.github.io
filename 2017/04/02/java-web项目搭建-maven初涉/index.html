maven简介及环境搭建

mvn简介

  • maven是基于项目对象模型（POM）,可以通过一小段描述信息来管理项目的构建，报告和文档的软件项目管理工具。强大的自动化构建工具（compile,clean,test….），提供仓库的工具，统一管理第三方jar包。

    mvn目录：

  • bin目录包含mvn运行脚本
  • boot 包含类加载器框架
  • conf 是配置文件目录(settings.xml)
  • lib 包含类库（自身及第三方）

maven环境变量配置,在命令行模式中输入mvn -v，如果有显示，则安装成功
